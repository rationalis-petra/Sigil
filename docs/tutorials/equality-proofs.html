The Equality Type

We want to make an equality type, letting is construct types like 2 ≡ 1 + 1`. This makes the definition of the type itself easy: for any type A and values x, y : A, we can construct the type x ≡ y.

However, we only want the type to be inhabited in the case where x and y are the same, so how do we do this? We do this by taking advantage of the reflexivity property of equality, i.e. any x is equal to itself. 

(induct (_≡_ : {A} → A → A → 𝒰)
  (refl : {A} → (x : A) → x ≡ x))

This means we can construct values of the type 3 - 1 ≡ 1 + 1 

(twoeq : 3 - 1 ≡ 1 + 1)
(twoeq ≜ refl 2)

Glyph will accept this code, because when comparing 2 and 1 + 1, it will first fully evaluate the term - as both 3 - 1 and 1 + 1 evaluate to 2, the type checker accepts our definition, and we have a proof that 3 - 1 equals 1 + 1.

Properties of Equality

Symmetry

(sym : {A} → {x y : A} → x ≡ y → y ≡ x)
(sym refl ≜ refl)

Transitivity

(trans : {A} → {x y z : A} → x ≡ y → y ≡ z → x ≡ z)
(trans refl refl ≜ refl)

Congruence

(cong : {A B} → {x y : A} → (f : A → B) → (x ≡ y) → (f x ≡ f y))
(cong f refl ≜ refl)

(refl x) → refl (f x)

Application: 0 as an Identity

(induct (ℕ : 𝒰)
  (zero : ℕ)
  (succ : ℕ → ℕ))

(_+_ : ℕ → ℕ → ℕ)
(n + m ≜ match n
  (zero → m)
  (suc k → suc (m + k)))

(+-zero-lid : (m : ℕ) → zero + m ≡ m)
(+-zero-lid m refl ≜ refl)

(+-zero-rid : (m : ℕ) → m + zero ≡ m)
(+-zero-rid m refl ≜ match m
  (zero → refl)
  (suc k → cong suc (+-zero-rid k)))
