Conjunction

  (⟨_,_⟩ : {A B} → A → B → A ∧ B))

(tandt ≜ ⟨◇, ◇⟩)

(tandf : True ∧ False)
(tandf ≜ ⟨◇, ???⟩)

Disjunction 

(induct _∨_ : (A : 𝒰) → (B : 𝒰) → 𝒰
  (left : {A B} → A → A ∧ B)
  (right : {A B} → B → A ∧ B))

(tort : True ∨ True)
(tort ≜ left ◇)

(torf : True ∨ False)
(tort ≜ left ◇)

(t-or-f-is-t : True ∨ False → True)

More General Proofs

if A ∧ B then B if A ∧ B then A 

(fst : {A B} → (A ∧ B) → A)
(fst ⟨l, _⟩ ≜ l)

(snd : {A B} → (A ∧ B) → B)
(snd ⟨_, r⟩ ≜ r)


;; Our earlier proof that True ∧ True ⇒ True is just a special case
(t-and-t-is-t : True ∧ True → True)
(t-and-t-is-t = fst {True} {True})

if (A ∨ B) and A → B and B → C then C 

(∨-elim : {A B C} → (A ∧ B) → (A → C) → (B → C) → C)
(∨-elim aorb atoc btoc ≜ match aorb
  (left a → atoc a)
  (right b → btoc b))

if (A ∨ A) then A 

(∨-simplify : {A} → (A ∧ A) → A)
(∨-simplify aora ≜ ∨-elim aora id id)

Universal Quantification

Useful Proofs

So far, we have shown only very simple proofs, like "A and B implies A". However, most mathematical proofs talk about some kind of object, like the natural numbers. A simple proposition on the naturals might be something like "for any number n we have that n + 0 = 0 + n = n". As a type, this might look something like: 

(0-id : (n : ℕ) → (n + 0 ≡ n) ∧ (0 + n ≡ n))

However, we see that to make this statement, we need a new type - an equality type (this type uses ≡ to distinguish it from the function '='). This is what will be discussed in the next tutorial.
