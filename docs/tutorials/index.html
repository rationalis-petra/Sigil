Quickstart Guide

The Quickstart Guide features a set of increasingly complex examples which can be easily copied into a local or online editor and run. While each example contains a brief description of the concepts in play, they are intended to demonstrate what is possible, rather than teach, so don't worry if some or most of the examples don't make sense/are difficult to understand. Programmers with experience of Dependently typed languages may find them a useful reference.

Language

The language tutorials come in three sets: Glyph, Seuth and Proof. All three are designed to be approachable both by new programmers and programmers who have experience in imperative programming, although the latter may find it prudent to skip some parts. Each tutorial has an accompanying set of exercises.

If you have experience in a similar language (Agda, Coq, Haskell, etc.) then it may be worth skimming the tutorials, but primarily relying on the documentation, referring back to tutorials when needed.

The Tutorials are in roughly linear order, with each building on ideas introduced in the last, so it is recommended you work slowly, and use the exercise sheets to solidify your understanding (if necessary).

Set 1: Glyph

This first set of tutorials lays out the language, including all concepts you'll need to write advanced, fully-featured programs. It is recommended you don't start on Set 2: Seuth or Set 3: Proofs until after finishing the Seuth Tutorial.

Set 2: Seuth

This second set teaches the Seuth language. Seuth is an auxiliary programming language, useful only in very specific contexts. It is considered an advanced feature, and a full understanding of Seuth is not needed to write good programs. It is therefore recommended you first skim the tutorials to decide if it is work learning before diving in.

Set 3: Proofs

Like many other dependently typed language, Glyph can be used to express and prove Mathematical theorems by exploiting the Curry-Howard correspondence. This set describes how prove theorems in Glyph. As with Set 2, it is best first considering if this aspect of Glyph is something you personally consider worth learning before diving in.
