(module main
  (import
    prelude
    sys.console.(get-line put-line)))

(greet : String → String)
(greet name ≜ "Hello, ${name}")

(main : IO Unit)
(main ≜ do
  (put-line "What is your name?")
  (name ← get-line)
  (put-line (greet name)))

What Is sigil?

sigil is a modern, powerful language drawing inspiration from Common Lisp, Haskell, Caledon and Agda. The goal of sigil is to provide a language which is:

  • Beautiful: The Semantics of sigil should be based on an elegant mathematical model, providing a precise interpretation and enabling meaningful abstractions.
  • Productive: sigil should be designed to enable an experienced developer to both make changes to existing code and write new code quickly and effectively.

For more info, check out our values section to see whether sigil's values align with yours. Some of its more notable features include:

  • Dependent Types
  • Implicit arguments
  • Typeclasses
  • Seuth, a Higher Order Logic Language
