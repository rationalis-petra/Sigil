Working with Unicode

sigil / Documentation / Unicode

sigil makes heavy use of Unicode characters in its' syntax. This is for two reasons:

  • To make code more terse - we can replace longer expressions like (defn even? [x] (equal (modulo x 2) 0)) with much shorter expressions (even? x ‚âú x | 2 = 0), which arguably convey the meaning at least as well if not better.
  • To write code which is clearer - this is particularly relevant when 'porting' symbols from Mathematics - we can write ‚â§, ‚â• and ‚â† instead of what many other languages use (<= >= and != or /=)
